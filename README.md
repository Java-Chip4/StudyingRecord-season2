## 레포지토리 소개
> Spring, JPA, Java 스터디 기록 레포지토리 입니다.
> 이슈에 학습 중 생긴 질문을 올리고 답변을 모두가 함으로써 깊은 토론이 될 수 있도록 합니다.

## 스터디 규칙
1. 매주 강의 분량을 정한다.
2. 정해진 분량의 강의를 학습하면서 질문을 최소 한개 이상씩  **`'이슈'`**에 올린다.
3. 이슈에 올라간 질문에 스터디원들은 답변을 달고 토의를 한다.
4. 주차별 이슈는 일요일 자정까지 올린다.
